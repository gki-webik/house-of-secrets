30 минут на кухне: кабачки в духовке с яйцом и сыром


- кабачки ( 800 гр. )
- молоко ( 200 мл. )
- твёрдый сыр ( 100 гр. )
- куриные яйца ( 4 шт. )
- растительное масло ( 1 ст. ложка )
- соль, перец, зелень ( по вкусу )

Приготовление:
1. Вымойте кабачки, если у них толстая шкурка и крупные семена – удалите их. Нарежьте кабачки шайбами толщиной три-четыре миллиметра.
2. Взбейте яйца с молоком, приправьте солью и перцем по вкусу.
3. Натрите сыр на крупной терке.
4. Форму для запекания смажьте растительным маслом. Выложите кабачки в несколько слоев.
5. Залейте кабачки омлетной массой, посыпьте тертым сыром.
6. Запекайте в разогретой до 180ºС духовке в течение 30 минут, до готовности кабачков. Время приготовления может меняться в зависимости от характеристик духовки.
7. При подаче можно украсить запеченные кабачки зеленью.
