Фриттата по испански


- Картофель ( 3 шт. )
- Лук ( 1 шт. )
- Болгарский перец ( 1 шт. )
- Яйцо ( 4 шт. )
- Ветчина ( 100 гр. )
- Сыр ( 30 гр. )
- Соль
- Перец

Приготовление:
1. Нарезать кольцами лук
2. Обжарить его на растительном масле
3. Нарезать кубиками перец, добавить к луку и тушить 5 минут
4. Добавить помидоры
5. Картофель отварить, очистить и нарезать кубиками
6. К перцу и луку добавить картофель и жарить до золотой корочки
7. Посолить и поперчить
8. Взбить яйца и добавить к ним ветчину
9. В форму для запикания выложить овощи
10. Влить яичную смесь на овощи
